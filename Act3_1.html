Mi portafolio

Simulación de almacén - Noviembre 2021

Realicé en Unity la simulación de 5 montacargas que recorrían los pasillos de un almacén. Codifiqué en C# un grafo que respresentaba el camino que los montacargas podían seguir y se establecía una ruta aleatoria para cada uno. Diseñé con la herramienta de ProBuilder, la estructura del almacén, los estantes y las cajas.

Prototipo de red de contenedores smart - Octubre a Noviembre 2021

Comenzamos utilizando un simulador de datos para practicar la conexión entre cliente y servidor, para posteriormente obtener datos de manera real con un sensor ultrasónico y guardarlos en una base de datos relacional. Contectamos el sensor a una Raspberry Pi, y lo conectamos mediante protocolo MQTT a nuestra base de datos para almacenar las mediciones. Obtuvimos información de la base de datos, que después mostramos visualmente en el dashboard de nuestro sitio web.

❮ ❯

Sitio web de filtros - Agosto 2021

Realicé un sitio web sencillo, donde utilicé JavaScript para programar filtros que son aplicables a una imagen que suba el usuario. El código toma pixel por pixel los valores RGB de la imagen original, para generar un nuevo pixel con los valores modificados y componer la imagen final. En el caso del filtro de distorción, el programa cambia en cierto rango la posición del pixel, a partir de la posición original que tenía.
